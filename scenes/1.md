__Syntax:__ `Kamera: Dir: {x,y}, start: {x,y}, end: {x,y}`
oder: `Kamera: Dir: {x,y}, pos: {x,y}`

Dir = wo hin zeigt sie, pos = position, start = start, end = end, dir2 = wohin sie sich dreht
